# 🥗 Asistente Nutricional IA

¡Bienvenido! Este proyecto es un **Asistente Nutricional inteligente** que utiliza modelos de IA para generar planes de alimentación personalizados.

## 🚀 Tecnologías y Aprendizaje
Proyecto desarrollado tras completar la formación de **Víctor Robles**:
* **Backend:** NodeJS.
* **IA:** Integración con APIs de OpenAI / DeepSeek.
* **Frontend:** JavaScript moderno (ES6+), HTML5 y CSS3.
